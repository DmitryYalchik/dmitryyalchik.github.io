Как использовать

Сгенерируйте бейдж с нужными параметрами и скопируйте код для вставки. Вставьте этот код в ваш README.md файл на GitHub:

  1. Настройте параметры бейджа: размер, цвета и текст
  2. Выберите стиль бейджа
  3. Нажмите "Сгенерировать бейдж"
  4. Скопируйте код из поля "Код для вставки"
  5. Вставьте код в ваш Markdown-файл на GitHub
  6. При необходимости скачайте бейдж для использования вне GitHub
